# Auto-Correlation Tool

This project will aim to produce a tool capable of ingesting data and searching for strong correlations between 
between variables. The code will cover file reading, data ingestion, data restructuring, statistical 
analysis, and outputs (graphical and tabular). 

## Getting Started

This application was created using Python 3.7.4 and requires the dependencies listed in requirements.txt.

### Prerequisites

This application requires Python 3 and the dependencies listed in requirements.txt. These Python modules should 
be installed within an active virtual environment using pip:

```
bash
pip install -r requirements.txt
```
